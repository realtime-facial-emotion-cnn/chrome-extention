# Summary Page README

- LLM call endpoint should be called here.
- From the popup window, required data for the LLM call is stored inside `localStorage`, so get the data from local storage.

This page is the summary page, the main page for the extension, so it should be created in an attractive way.

## Endpoint Data

The following fields are received from the endpoint and can be arranged in an attractive way for the user:

- `grammar_score`
- `fluency_score`
- `confidence_score`
- `emotion_observation`
- `strengths`
- `improvements`
